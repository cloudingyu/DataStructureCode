# 最大子段和

## 题目背景

给出一个长度为 $ n $ 的整数序列 $ a_1, a_2 \cdots a_n $。

你需要找一个子区间 $[l, r] (1 \leq l \leq r \leq n)$，使得 $ \sum_{i=l}^{r} a_i $ 尽可能大。

你只需要输出这个最大值即可。

## 输入格式

第一行输入一个整数 $ n $，表示序列长度。

第二行 $ n $ 个整数，$ a_1, a_2 \cdots a_n $。

## 输出格式

输出一行，一个整数，表示答案。

## 样例 #1

### 样例输入 #1
```
9
3 -1 -4 1 -5 9 -2 6 -5
```

### 样例输出 #1
```
13
```

## 数据范围与提示

对于 100% 的测试数据满足 $ 1 \leq n \leq 10^5 $，$ -10^9 \leq a_i \leq 10^9 $ 。
