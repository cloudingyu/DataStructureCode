## Description

所谓后缀表达式是指这样的一个表达式：式中不再引用括号，运算符号放在两个运算对象之后，所有计算按运算符号出现的顺序，严格地由左而右新进行（不用考虑运算符的优先级）。

如：`34*(5-2)+7` 对应的后缀表达式为：`34. 5. 2. -*7. +@`。`@` 为表达式的结束符号。`.` 为操作数的结束符号，防止操作数连在一起分不清。

## Format

### Input

输入：后缀表达式，以 `@` 结尾

### Output

输出：表达式的值，运算结果

## Samples

### 输入样例 1

```
3.5.2.-*7.+@
```

### 输出样例 1

```
16
```

## Limitation

字符串长度，1000内。

运算符只有加减乘除四种(`+-*/`)，保证所有的中间结果都是 `int` 范围
