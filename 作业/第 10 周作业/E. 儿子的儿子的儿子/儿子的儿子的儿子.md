# 儿子的儿子的儿子......

## 题目描述

Y 虽然让同学们站成一棵树的样子，但是同学们都不安分。

每个同学都想知道自己的儿子的儿子的儿子......有多少个。

即求，对于每个节点 $v$，在其子树中距离自己恰好为 $k$ 的节点有多少个。

## 输入格式

第一行两个数 $n$ 和 $k$。

第二行 $n - 1$ 个整数分别表示第 $2, 3, \cdots , n$ 号同学的“父亲”。

## 输出格式

输出 $n$ 个数空格隔开表示每个同学的儿子的儿子的儿子......的数量。

## 样例 #1

### 样例输入 #1

```
5 1
1 1 2 1
```

### 样例输出 #1

```
3 1 0 0 0
```

## 数据范围与提示

$1 \le n, k \le 5 \times 10^6$。

节点编号从 1 开始，且 1 为根。
