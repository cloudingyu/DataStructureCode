# 逆序对

## 题目描述

ZZR 有一个序列 $a_1, a_2, \cdots, a_n$，他允许最多进行 $k$ 次操作，每次操作交换两个相邻元素。

求经过变换后的序列中最少还有多少逆序对。

逆序对指的是二元组 $(i, j)$，其满足 $i < j$ 且 $a_i > a_j$。

## 输入格式

第一行包含两个数 $n, k$，表示序列长度和交换两个相邻元素的次数上限。

第二行有 $n$ 个数，表示序列 $a_1, a_2, \cdots, a_n$。

## 输出格式

一个数表示最少的逆序对的数量。

## 样例 #1

### 样例输入 #1

```
3 1
2 2 1
```

### 样例输出 #1

```
1
```

## 样例 #2

### 样例输入 #2

```
3 0
2 2 1
```

### 样例输出 #2

```
2
```

## 数据范围与提示

$1 \le n \le 10^5$, $0 \le k \le 10^9$, $1 \le a_i \le 10^9$。
