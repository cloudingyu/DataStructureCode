# 血魔训练 II

## 题目描述

ZZR 经常玩 DOTA2，尤其喜欢玩血魔。现在 ZZR 正在练习玩血魔。

血魔的技能之一为，每当他击杀一个英雄单位后，他能够回复该英雄生命值上限百分比的血量。在此题中：

假设 ZZR 有 $x$ 滴血，现在有 $n$ 个敌方英雄。ZZR 杀死每个敌方英雄需要损失 $a_i$ 滴血，但是当他杀死该英雄之后能够恢复 $b_i$ 滴血。

当 ZZR 血量小于或等于 0 时 ZZR 就阵亡，训练结束。

ZZR 想要击杀尽可能多的敌方英雄，请问他在保证自己不阵亡的情况下，能否击杀所有英雄。

## 输入格式

第一行包括两个数 $n, x$，分别表示 $n$ 个英雄和 $x$ 滴血。

第二行包括 $n$ 个数 $a_1, \cdots, a_n$，表示 ZZR 击杀每个英雄所需要消耗的血量。

第三行包括 $n$ 个数 $b_1, \cdots, b_n$，表示 ZZR 击杀每个英雄恢复的血量。

## 输出格式

'YES' 或 'NO'，表示能否击杀所有英雄。

## 样例 #1

### 样例输入 #1

```
4 10
12 11 2 3
10 2 5 4
```

### 样例输出 #1

```
YES
```

## 样例 #2

### 样例输入 #2

```
3 10
6 8 7
2 6 4
```

### 样例输出 #2

```
NO
```

## 样例 #3

### 样例输入 #3

```
2 14
12 11
2 10
```

### 样例输出 #3

```
YES
```

## 数据范围与提示

$1 \le n \le 10^6$, $1 \le x, a_i, b_i \le 10^9$。

**样例 1 解释**: ZZR 首先击杀第 3 个英雄，还剩 13 滴血，再击杀第 4 个英雄，还剩 14 滴血，再击杀第 1 个英雄，还剩 12 滴血，最后击杀第 2 个英雄，还剩 3 滴血。如果 ZZR 在击杀第 3、4 个英雄后先击杀第 2 个英雄则剩 3 滴血，则剩余血量就无法击杀第 1 个英雄。
