# 本质不同子串计数

## 题目背景

给出一个字符串 $S$，请你统计其中本质不同的子串个数。

**例如**：如果 $S = \text{aba}$，那么它有 6 个子串，`a,b,a,ab,ba,aba`。但是，本质不同的子串只有 `a,b,ab,ba,aba` 这 5 种。

## 输入格式

第一行，一个字符串 $S$。

## 输出格式

输出一行，一个整数表示 $S$ 的本质不同的子串个数。

## 样例 #1

### 样例输入 #1

```
aba
```

### 样例输出 #1

```
5
```

## 样例 #2

### 样例输入 #2

```
george
```

### 样例输出 #2

```
18
```

## 数据范围与提示

$|S| \le 2000$，输入字符串均由小写字母组成。
