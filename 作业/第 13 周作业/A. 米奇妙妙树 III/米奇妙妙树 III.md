# 米奇妙妙树 III

## 题目描述

Y 非常喜欢米奇妙妙屋，于是他在自己家里种了一棵有 $n$ 个节点的米奇妙妙树。

Y 还养了一条名为布鲁托的狗，这条狗总是叼着一把刷子给树染上不同的颜色。

已知树上每个节点最初的颜色是 $0$，节点 $1$ 为树的根，布鲁托会进行 $q$ 次染色操作，每次操作把节点 $x$ 到节点 $y$ 的路径上的所有节点（包括 $x$ 和 $y$ 本身）都染成颜色 $c$。

请问最后每个节点的颜色是什么。

## 输入格式

第一行一个整数 $n, q$ 分别表示节点数量和操作数量。

接下来 $n - 1$ 行，每行两个整数 $u, v$，表示一条边。

接下来 $q$ 行每行三个数 $x, y, c$ 表示这 $q$ 个操作。

## 输出格式

一行 $n$ 个数用空格隔开表示每个节点的颜色。

## 样例 #1

### 样例输入 #1

```
5 4
1 2
1 3
2 4
3 5
3 1 2
3 2 4
4 1 1
2 3 5
```

### 样例输出 #1

```
5 5 5 1 0
```

## 数据范围与提示

$1 \le n, q \le 10^6$, $1 \le x, y \le n$, $1 \le c \le 10^9$, $x \ne y$.
