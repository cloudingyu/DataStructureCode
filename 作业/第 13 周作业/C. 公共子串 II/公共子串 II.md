# 公共子串 II

## 题目描述

给定两个字符串 $s$ 和 $t$，计算字符串 $s$ 和 $t$ 公共子串数量。

假设第一个字符串 $s = (s_1, s_2, ..., s_n)$ 和 $t = (t_1, t_2, ..., t_m)$，统计有多少对 $(i, j, k)$ 使得子串 $s_i, ..., s_{i+k-1}$ 和 $t_j, ..., t_{j+k-1}$ “相等”。（其中 $k > 0, 1 \le i + k - 1 \le n, 1 \le j + k - 1 \le m$）。

这里我们定义两个字符串 $s$ 和 $t$ 是相等当且仅当将两个字符串按照字典序排序之后，每一个位置都对应相等，例如字符串 $abc$ 和字符串 $acb$ 在这种定义下被认为是相等的。

## 输入格式

第一行两个整数 $n$ 和 $m$ 表示字符串长度。

第二行为长度为 $n$ 的字符串 $s$ 和长度为 $m$ 的字符串 $t$。

两个字符串之间用空格隔开。

## 输出格式

一个整数，表示字符串 $s$ 和 $t$ 公共子串对数量。

## 样例 #1

### 样例输入 #1

```
7 7
strange egnarts
```

### 样例输出 #1

```
28
```

## 数据范围与提示

$2 \le n, m \le 3000$.
