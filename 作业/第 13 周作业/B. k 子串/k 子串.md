# k 子串

## 题目描述

给定一个长度为 $n$，且全部为小写字母的字符串 $s$。

定义 $k$ 子串为字符串 $subs_k = s_k s_{k+1} ... s_{n+1-k}$，显然 $subs_1 = s$，且有 $\lceil \frac{n}{2} \rceil$ 个 $k$ 子串。

当满足以下条件时，我们称字符串 $t$ 为 $T$ 的奇前后缀。
- $|T| > |t|$
- $|t|$ 是一个奇数
- $t$ 同时是 $T$ 的前缀和后缀

现在对于每一个 $k$ 子串 ($k \in [1, \lceil \frac{n}{2} \rceil]$)，求它的最大奇前后缀的长度。

## 输入格式

第一行一个整数 $n$，表示字符串 $s$ 的长度。

第二行为字符串 $s$，包含 $n$ 个小写字母。

## 输出格式

输出 $\lceil \frac{n}{2} \rceil$ 个数，第 $i$ 个数表示 $i$ 子串的最大奇前后缀的长度。如果没有最大奇前后缀，则输出 $-1$。

## 样例 #1

### 样例输入 #1

```
15
bcabcabcabcabca
```

### 样例输出 #1

```
9 7 5 3 1 -1 -1 -1
```

## 样例 #2

### 样例输入 #2

```
24
abaaabaaaabaaabaaaabaaab
```

### 样例输出 #2

```
15 13 11 9 7 5 3 1 1 -1 -1 1
```

## 样例 #3

### 样例输入 #3

```
19
cabcabbcbcabbcbcabca
```

### 样例输出 #3

```
5 3 1 -1 -1 1 1 -1 -1 -1
```

## 数据范围与提示

$1 \le n \le 10^6$

第一个样例：
1子串：bcabcabcabcabca
2子串：cabcabcabcabc
