# G. k一串的几何题

## 题目描述

kjhhjki 今天遇到了一道几何题，它定义了二维平面上一条直线 $l$ 关于两个点 $A, B$ 的曲值如下：

$$
val(l, A, B) = \max_{X \in l} |AX - BX|
$$

现在他有两个点 $A, B$ 的坐标分别为 $(c, 0), (-c, 0)$ 和 $n$ 个两两不同的点，求所有 $\frac{n(n-1)}{2}$ 条直线中 $val(l, A, B)$ 的最小值。

## 输入格式

第一行输入两个整数 $n, c$ ($n \leq 2 \times 10^5$, $1 \leq c \leq 10^5$)，表示点的数量和特殊点位置。

接下来 $n$ 行，每行两个整数 $x, y$ ($-10^5 \leq x, y \leq 10^5$)，表示一个点的坐标。

## 输出格式

输出一个有理数 $v$，表示答案。你的答案的相对误差或绝对误差中有一个小于 $10^{-6}$ 即为正确答案，即满足 $\frac{|v - v_{ans}|}{v_{ans}} \leq 10^{-6}$。

## 样例 #1

### 样例输入 #1

```
2 5
1 0
2 1
```

### 样例输出 #1

```
7.2111025509
```

## 样例 #2

### 样例输入 #2

```
3 6
0 1
2 5
0 -3
```

### 样例输出 #2

```
0.0000000000
```
