# 选课

## 题目描述

3202年，FDUCS的同学们已经卷上天啦！现在一共有 $n$ 门课要修，而学校为了让同学们不再为选课踏心，每门课的学分都变成相同的了。

现在一门课的绩点上限已经不再是 4.0 了，因为这已经完全满足不了同学们卷成绩了。具体来说，现在每门课的绩点最高为 $m$，且一个同学的绩点必须是 $1, 2, 3, \cdots, m$ 这些整数中的一个。

此外，学校为了防止同学们乱选课，给出了 $k$ 条选课限制，每个限制表示为 $u, v (1 \leq u, v \leq n)$，意思是必须先修完课程 $u$ 才能修课程 $v$。

而且现在每个学期能且只能修一门课，因此不能够同时上好几门课，必须等到一门课修完了才能修下一门课。也不允许在某个学期不修课程，这太摆烂了。

现在，小聂同学已经根据超大模型预测出了自己每门课的绩点，第 $i (1 \leq i \leq n)$ 门课的绩点将会是 $p_i$，且不同课程的绩点两两不同（$\forall 1 \leq i < j \leq n, p_i \neq p_j$）。

请你帮小聂同学规划一个最优的选课顺序，使得他满足学校的 $k$ 条选课限制，并且每个学期出成绩后，他的 GPA（平均绩点，即所有已修课程的绩点平均值）要尽可能高，这样才能有可能卷赢其他同学。

## 输入格式

第一行，3个整数 $n, m, k$，分别表示课程总数，每门课的绩点上限，学校的选课限制数量。

第二行，$n$ 个整数 $p_1, p_2, \cdots, p_n$，其中 $p_i (1 \leq i \leq n)$ 表示小聂同学修完课程 $i$ 后这门课的绩点。保证小聂同学不同课程的绩点互不相同。

接下来 $k$ 行，每行两个整数 $u, v (1 \leq u, v \leq n)$ 表示必须先修完课程 $u$ 之后才能修课程 $v$。

数据保证一定存在至少一个合法顺序可以修完所有课程。

## 输出格式

输出一行，$n$ 个整数用空格隔开，从左到右表示按先后顺序选课。你需要保证按该顺序选课后，每个学期的 GPA 都尽可能高。（这意味着，如果在第 $i$ 个学期小聂同学有多门课可以选择，他总是会贪心选择能收获绩点最高的课程）。

## 样例 #1

### 样例输入 #1

```
5 5 4
1 2 3 4 5
2 1
1 3
1 4
2 4
```

### 样例输出 #1

```
5 2 1 4 3
```

## 数据范围与提示

- $1 \leq n \leq 2000$
- $1 \leq k \leq 4000$
- $1 \leq m \leq 10^6$

数据不保证同一条限制不会重复出现，即在 $k$ 条选课限制中存在重复出现的可能。
