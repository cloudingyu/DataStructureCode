# 二进制大数乘法
## 题目背景

输入两个很大的整数，你需要输出它们的乘积。

特别的，你需要完成 **2 进制** 下的这个问题，而非我们常用的 10 进制。

## 输入格式

- 第一行输入一个二进制串，保证最高位为 `1`。
- 第二行输入一个二进制串，保证最高位为 `1`。

## 输出格式

输出一行，一个二进制串，表示输入的两个二进制串相乘后的答案。

## 样例 #1

### 样例输入 #1

```
10
11
```

### 样例输出 #1

```
110
```

## 样例 #2

### 样例输入 #2

```
101
111
```

### 样例输出 #2

```
100011
```

## 数据范围与提示

对于 100% 的测试数据满足输入的二进制串长度不超过 5000。
