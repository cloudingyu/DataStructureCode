# 树德遍历

## 题目描述

SDZX 可以看作是一棵树，他有 $n$ 个结点，有 $n - 1$ 条边将他们相连。QiangBro 开着卡丁车，想把每个结点都访问至少一次，但是他每经过一条边，都会消耗1块钱的电费！

请为他规划起点和开车的路线，使得拜访每个结点至少一次，且消耗尽可能少的电。

输出他最少需消耗的电费。

## 输入格式

第一行，一个正整数 $n$。

接下来 $n - 1$ 行，每行两个整数 $u, v$，表示结点 $u, v$ 之间存在一条边将他们相连。

保证给出的边构成一棵合法的树。

## 输出格式

输出一行，一个整数，最少的电费开销！

## 样例 #1

### 样例输入 #1

```
5
1 2
1 3
1 4
1 5
```

### 样例输出 #1

```
6
```

## 数据范围与提示

$1 \le n \le 10^5$
