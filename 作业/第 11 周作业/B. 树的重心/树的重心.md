# 树的重心

## 题目描述

树的重心被定义为：树中的一个节点，删除该点后，使得形成的多棵树中节点数最大的树的节点数最小。求树的重心。

## 输入格式

第一行包含一个整数 $n$，表示树节点的个数。

接下来 $n - 1$ 行，每行两个整数，表示一条边的两个节点。

节点编号从 $1$ 到 $n$。

## 输出格式

输出树的重心。如果有多个重心，则按编号从小到大依次输出。

## 样例 #1

### 样例输入 #1

```
6
1 2
2 3
2 5
3 4
3 6
```

### 样例输出 #1

```
2 3
```

## 数据范围与提示

$1 \le n \le 5 \times 10^4$
