# 本质不同子串计数

## 题目描述

哗啦啦，哗啦啦，天上掉落一条随机字符串。

萌哒哒，萌哒哒，兔兔数起本质不同子串数。

## 输入格式

输入一个仅包含小写字母的字符串 $s$，保证 $1 \le |s| \le 10^5$，且每个字符在 26 个小写字母中随机选取。

## 输出格式

输出一个正整数，表示字符串 $s$ 中本质不同的子串个数。

## 样例 #1

### 样例输入 #1

```
ababa
```

### 样例输出 #1

```
9
```

## 样例 #2

### 样例输入 #2

```
decbccedcbeecdb
```

### 样例输出 #2

```
107
```

## 数据范围与提示

$1 \le |s| \le 10^5$ ，时间限制：$1000\text{ms}$ ，空间限制：$512\text{MB}$
