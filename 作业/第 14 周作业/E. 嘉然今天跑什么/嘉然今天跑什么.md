# 嘉然今天跑什么

## 题目描述

嘉然上班要迟到了！

嘉然需要从一张 $n$ 个点 $m$ 条边的无向图上从 1 号点跑到 $n$ 号点。每条边有两个权值 $t$ 和 $w$ 分别表示路过的时间和过路费。

嘉然希望选择一条时间最短的路径，同时，鲁迅曾经说过：“有钱不省是 **。”，嘉然希望在时间最短的情况下选择一条过路费最小的。

请输出最小时间和过路费。

## 输入格式

第一行两个整数 $n$ 和 $m$ 分别表示点数和边数。

接下来 $m$ 行每行 4 个整数 $x, y, t, w$ 表示节点 $x$ 和节点 $y$ 之间有一条通过时间为 $t$，过路费为 $w$ 的路径。

## 输出格式

一行两个整数表示最小时间和过路费。

## 样例 #1

### 样例输入 #1

```
3 4
1 2 2 1
2 3 1 2
1 3 3 5
2 3 1 1
```

### 样例输出 #1

```
3 2
```

## 数据范围与提示

$1 \le n \le 10^5, 1 \le m \le 5 \times 10^5, 1 \le t, w \le 10^9$。

保证 1 号点一定能到 $n$ 号点。
