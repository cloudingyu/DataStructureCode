# 区间乘积的分类

## 题目描述

这是一道送分题。

有一个长度为 $n$ 的整数序列 $a_1, a_2, ...a_n$

接下来你需要求三个值：

1. 有多少个点对 $(l, r)$，满足 $l \le r$ 且 $a_l \cdot a_{l+1} \cdots a_{r-1} \cdot a_r < 0$
2. 有多少个点对 $(l, r)$，满足 $l \le r$ 且 $a_l \cdot a_{l+1} \cdots a_{r-1} \cdot a_r = 0$
3. 有多少个点对 $(l, r)$，满足 $l \le r$ 且 $a_l \cdot a_{l+1} \cdots a_{r-1} \cdot a_r > 0$

## 输入格式

第一行，正整数 $n$。

接下来一行，$n$ 个整数。

## 输出格式

一行三个整数，用空格隔开，分别对应题目描述中的三个问题的答案。

## 样例 #1

### 样例输入 #1

```
5
5 -3 3 -1 0
```

### 样例输出 #1

```
6 5 4
```

## 样例 #2

### 样例输入 #2

```
10
4 0 -4 3 1 2 -4 3 0 3
```

### 样例输出 #2

```
12 32 11
```

## 样例 #3

### 样例输入 #3

```
5
-1 -2 -3 -4 -5
```

### 样例输出 #3

```
9 0 6
```

## 数据范围与提示

对30%的数据满足：$n \le 1000$

对100%的数据满足：$1 \le n \le 200000$, $-10^9 \le a_i \le 10^9$
