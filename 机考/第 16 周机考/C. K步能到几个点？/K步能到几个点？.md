# K步能到几个点？

## 题目描述

给出 $n$ 个结点，以及 $n \times n$ 的 01 矩阵 $W_{n \times n}$，其中 $W_{i,j} = 1$ 则表示存在从 $i$ 走向 $j$ 的边；而 $W_{i,j} = 0$ 则表示不存在从 $i$ 走向 $j$ 的边。特别的，如果 $W_{i,i} = 1, (1 \le i \le n)$ 则表示存在从 $i$ 走到 $i$ 自己的自环。

现在，有 $m$ 次询问，其中第 $i$ 次询问会给出两个值 $x_i, k_i$，询问的内容是，如果从 $x_i$ 结点出发，走恰好 $k_i$ 步（每步经过一条可以走的边），请问最后停下来时所在的结点有多少种不同的可能？

## 输入格式

第一行，2 个整数 $n, m$，分别表示结点数量和问询数量。

接下来 $n$ 行，每行 $n$ 个整数（0 或 1），用空格隔开，表示矩阵 $W_{n \times n}$。其中第 $i$ 行第 $j$ 列的整数表示 $W_{i,j}$。

接下来 $m$ 行，每行两个整数 $x_i, k_i$ 表示一次问询。

## 输出格式

输出 $m$ 行，回答每组问询的答案。

## 样例 #1

### 样例输入 #1

```
3 3
1 1 0
1 1 0
1 1 1
1 3
2 2
3 1
```

### 样例输出 #1

```
2
2
3
```

## 数据范围与提示

$1 \le n \le 70$, $1 \le m \le 5000$, $1 \le x_i, k_i \le n$。
