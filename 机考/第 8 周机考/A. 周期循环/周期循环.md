# 周期循环

## 题目描述

ZZR 喜欢周期循环，看到不是周期循环的字符串就很不爽。现在 ZZR 得到一个仅含小写字母的字符串，你可以在该字符串末尾添加若干任意字符。请计算最少需要添加多少个字符，才能使字符串变为周期循环（存在一个循环节且至少循环两次）。

## 输入格式

第一行输入一个整数 T，表示测试数据组数。  
接下来 T 行，每行一个由小写字母组成的字符串，记其长度为 L。

## 输出格式

对于每组数据，输出一行一个整数，表示最少需要添加的字符数量。

## 样例

样例输入
```
3
ppp
pip
machinelearning
```

样例输出
```
0
1
15
```

样例解释
- ppp 已经是周期循环（循环节为 p，循环 3 次），无需添加。

## 数据范围与说明

- 1 ≤ T ≤ 100
- 3 ≤ L ≤ 1e5
