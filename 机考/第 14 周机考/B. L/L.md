# L

## 题目描述

要求在一个 $n \times m$ 的由小写字母构成的矩阵中找出一个 $L$ 形的由小写字母构成字符串的出现次数。一个长度为 $k$ 的 $L$ 形字符串长宽均为 $k$，粗细为 $1$，按先从上到下，再从左到右的顺序输入。如一个长度为 $3$ 的 $L$ 形字符串，其格式形如：

```
a
b
cde
```

## 输入格式

第 $1$ 行输入字符矩阵的长 $n$，宽 $m$，$L$ 形字符串的长度 $k$，$(n, m \le 5000, 2 \le k \le \min(n, m))$。第 $2$ 行到第 $n + 1$ 行输入 $n$ 个长为 $m$ 的小写字母字符串，表示一个 $n \times m$ 的矩阵。第 $n + 1$ 行输入长为 $2k - 1$ 的小写字母字符串，表示一个长度为 $k$ 的 $L$ 形字符串。

## 输出格式

一个数字，表示 $L$ 形字符串在矩阵中的出现次数。

## 样例 #1

### 样例输入 #1

```
5 4 3
bbbb
bbfd
dddd
rdfg
rdfg
bbddd
```

### 样例输出 #1

```
2
```

## 样例 #2

### 样例输入 #2

```
4 5 3
aedcf
bbbbf
cdeaf
fffff
abcde
```

### 样例输出 #2

```
1
```
